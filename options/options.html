Settings

Thanks for installing YouTooltip.

YouTooltip can reveal what a YouTube video or playlist link exactly is at a glance. YouTooltip also helps you save bandwidth usage, which is especially important on a metered network. In addition, you can also avoid clicking on common Rickroll videos. (Or you can click on them anyway. I don't care.)

Permissions

To get the most out of YouTooltip, it is recommended to grant access to all websites. You can do so by clicking the button below.

Granted

To activate YouTooltip without access to all websites, you can:

  • Click the toolbar button, which will run YouTooltip only for that page.
  • Allow access only to that site, which will run YouTooltip on every visit.

v1.6.0 (Apr 2, 2023)

  • Changed: Other options UI improvements.
  • Changed: YouTooltip now works on Youtube.
  • Changed: Code cleanup.
  • Fixed: Error when moving cursor off link when no custom tooltip exists.

  • Fixed: Error when text node changes, but not an element.
v1.5.0 (Jan 25, 2023)
  • Changed: Upgraded to Manifest V3. As a consequence, the minimum Firefox version is now 109.
  • Changed: Popup now highlights known Rickrolls in yellow.
  • Fixed: Console error when document head does not exist.
v1.4.0 (Dec 20, 2022)
  • Added: Welcome page. Automatically opens on install. Thereafter, the options page opens to the settings tab.
  • Added: Piped API service.
  • Added: YouTube Shorts detection.
  • Changed: YouTooltip no longer works on Cloudtube and Invidious instances.
  • Changed: Requests are now made in the background instead of the page in preparation for Manifest V3.
  • Changed: Youtube info is no longer stored using HTML data-* attributes when using hover operation mode and notification and custom tooltip display modes.
  • Fixed: Newly added links are sometimes skipped.
  • Fixed: Popup correctly states that YouTooltip does not work on Youtube, instead of perpetually waiting for page.
  • Fixed: Custom tooltip is hidden on some sites.

  • Added: Toolbar icon badge.
  • Added: Blacklist.

  • Changed: Statistics are no longer recorded in Private Browsing/Incognito tabs and windows.
  • Changed: Significant UI improvements for both options and popup.
  • Changed: Minimum Firefox version is now 98.
