Settings

Thanks for installing YouTooltip.

YouTooltip can reveal what a YouTube video or playlist link exactly is at a glance. YouTooltip also helps you save bandwidth usage, which is especially important on a metered network. In addition, you can also avoid clicking on common Rickroll videos. (Or you can click on them anyway. I don't care.)

Permissions

This section is only relevent when YouTooltip is using Manifest V3. In Manifest V2, these permissions are granted on install, and you do not need to do anything.

To get the most out of YouTooltip, it is recommended to grant access to all websites. You can do so by clicking the button below.

Granted

To activate YouTooltip without access to all websites, you can:

  • Click the toolbar button, which will run YouTooltip only for that page.
  • Allow access only to that site, which will run YouTooltip on every visit.
